# Gypsy

To test:

1. npm install
2. make sure truffle config file is configured to your local blockchain correctly
3. start your local blockchain
4. truffle migrate
5. npm run start
6. login to metamask
7. connect account from ganache
8. test functions

After modifying:

Smart contracts:
 1. truffle compile
 2. truffle migrate --reset
 3. npm run start

Javascript:
  should reload automatically
